### 🧰 System Requirements

* Android Studio
* Kotlin

### 🛠️ Installation 

#### Open in Android Studio, Get from Version Control -> enter repository URL.

```
https://github.com/Padman83/Tap_Counter.git
```

#### Or Use the Terminal.

```
git clone https://github.com/Padman83/Tap_Counter.git
```
